# Screenshots Directory

Place your demo assets here:

## Required Files

1. **demo.gif** - Animated demo showing ticket arrival → notification
   - Recommended: 30-60 seconds
   - Resolution: 800x600 or similar
   - Shows: Zendesk ticket created → ZenNotif popup notification → sound plays

2. **screenshot-dashboard.png** - Dashboard/settings view
   - Resolution: 400x600 (popup size)
   - Shows: Connected status, monitored views, settings panel

3. **screenshot-setup.png** - Initial setup screen
   - Shows: Subdomain input, connect button

4. **screenshot-notification.png** - Browser notification example
   - Shows: Chrome notification with ticket details

## How to Create Demo GIF

### Option 1: Screen Recording → GIF
1. Use OBS or ScreenToGif to record your screen
2. Show: Creating a test ticket in Zendesk → ZenNotif alert appearing
3. Convert to GIF using ScreenToGif or ezgif.com

### Option 2: Browser Extension
- Chrome: Chrome Capture (screencastify is paid)
- Firefox: Built-in screenshot tool

## Optimization Tips

- Keep GIF under 2MB for fast loading
- Use 10-15fps for smooth but compact GIFs
- Crop to relevant area only
- Consider WebM for better quality (GitHub supports it)
