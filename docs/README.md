# ZenNotif GitHub Pages

This folder contains the landing page for ZenNotif, hosted on GitHub Pages.

## URL

Live site: `https://username.github.io/zennotif/`

## Structure

- `index.html` - Main landing page
- `styles.css` - Dark theme styling
- `script.js` - Interactive demos
- `screenshots/` - Demo assets

## Features

- 🌙 Dark theme design
- 🎮 Interactive demo (simulate ticket arrival)
- 🔊 Tone preview buttons
- 📱 Responsive layout
- ⚡ Smooth animations

## Development

```bash
# Serve locally
npx serve .

# Or use Python
python -m http.server 8000
```

## Deployment

GitHub Pages is configured to serve from the `/docs` folder on the main branch.

1. Go to repo Settings → Pages
2. Source: Deploy from a branch
3. Branch: main /docs folder
4. Save

## Customization

Update these in `index.html`:
- GitHub username in links
- Chrome Web Store link (when published)
- Screenshots/demo GIF
